## 概述

LINE 是 Kairos 支持的消息平台之一，在日本尤为流行。你可以直接在 LINE 中与 Kairos 交互：接收通知、发送指令、点击操作按钮（Flex Messages），一个聊天窗口搞定所有事。

## 设置方法

1. 登录 Kairos 控制台，选择 **LINE** 作为你的消息平台
2. 扫描二维码添加 Kairos LINE 官方账号，或直接点击链接
3. 在 LINE 中打开 Kairos 账号，发送控制台上显示的**配对码**
4. 验证通过后，Kairos 会自动为你激活专属沙盒环境

配对码有效期为几分钟，过期后可以在控制台点击**刷新**重新生成。

<Tip>
  也可以直接访问 [https://lin.ee/XSVJ0SY](https://lin.ee/XSVJ0SY) 添加 Kairos LINE 官方账号。
</Tip>

## 你可以做什么

通过 LINE，你能使用 Kairos 的全部功能：

- **接收通知**：新邮件、品牌合作、日程变更、付款提醒等
- **点击操作按钮**：通过 Flex Messages 评估合作、回复邮件、创建日程等
- **自然语言对话**：直接用文字提问、下达指令、描述合作需求
- **转发消息**：把其他聊天中的消息转发给 Kairos 处理
- **发送截图**：私信截图、个人资料、合同内容，Kairos 都能识别
- **设置偏好和规则**：告诉 Kairos 你的习惯，它会记住并执行

## 安全性

你的 LINE 账号通过配对码验证与 Kairos 账户绑定。只有完成配对的用户才能与 Kairos 交互，所有消息都在独立的沙盒环境中处理。

## 断开连接

如需停止使用，前往 Kairos 控制台断开 LINE 连接即可，沙盒环境和相关数据会被一并清除。

你也可以在 LINE 中直接屏蔽 Kairos 官方账号。如果之后重新添加并配对，将会是一个全新的环境，不保留任何历史数据。
