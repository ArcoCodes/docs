## 在 Telegram 里搜索网络

Kairos 内置网络搜索——不需要 API Key，不需要任何设置和额外工具。直接问问题，你的 AI 经纪人就去找答案。

## 如何使用

像平时搜索一样，直接问：

- _"科技类 YouTube 赞助的平均报价是多少？"_
- _"VidCon 2026 什么时候举办？"_
- _"FTC 对赞助内容披露有哪些规定？"_
- _"帮我找一些好的品牌合作媒体资料包案例"_

Kairos 搜索网络后给你一个简洁实用的答案——不只是一堆链接。

## 为什么好用

### 开箱即用

网络搜索无需配置，直接可用，不需要连接或设置任何东西。

### 新闻与时效性查询

需要最新信息？Kairos 可以按时间范围筛选结果：

- 品牌的最新动态
- 行业近期趋势
- 当前市场报价参考
- 即将举办的活动和日期

### 感知上下文

在合作评估或谈判过程中搜索时，Kairos 会把网络搜索结果与你的合作背景结合起来。例如，谈判期间搜索 _"Instagram Reels 平均报价"_，它会综合你的创作者层级和具体合作条款来解读结果。

## 不只用于品牌合作

网络搜索不限于合作场景，你可以用它做任何事：

- 内容选题调研和创意参考
- 行业新闻和趋势追踪
- 核实品牌方的说法
- 寻找工具、模板或资源
