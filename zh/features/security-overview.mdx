## 你的数据，你做主

我们知道你在把敏感信息托付给 Kairos——邮件、品牌合作、合同、财务细节。我们认真对待这份信任。

## 我们如何保护你的数据

### 隔离环境

每位用户都拥有完全私有的沙箱环境。你的数据、认证凭据和 AI 经纪人状态与其他所有用户完全隔离。没有共享存储，没有跨用户访问。

### 加密

所有认证凭据（Gmail 令牌、Notion 令牌、机器人令牌）均采用 **AES-256-GCM** 加密——与银行和金融机构使用的相同加密标准。凭据在静态时加密存储，仅在需要执行授权操作时才会解密。

### 邮件隐私

新邮件到达时，Gmail 只向 Kairos 发送一个通知和邮件 ID——不包含邮件内容本身。Kairos 将这个 ID 转发到你的私有环境，由你的 AI 经纪人直接从 Gmail 读取邮件。我们的服务器仅存储一个短 ID 以避免重复处理，邮件主题、正文、附件和发件人信息从不进入我们的数据库。

### Telegram 安全

你的 Telegram 机器人由你通过 BotFather 创建并拥有。Kairos 在每次操作时都通过机器人的所有者 ID 验证你的身份——即使在群聊中，其他人也无法与你的 AI 经纪人互动。

## Kairos 能做什么，不能做什么

| | 详情 |
|---|---|
| **可以读取** | 邮件（Gmail）、日历事件、你授权访问的 Notion 页面 |
| **可以创建** | 日历事件、Notion 页面、邮件草稿 |
| **可以发送** | 邮件——仅在你明确确认后 |
| **不能访问** | Google 云端硬盘、Google 相册，以及任何你未连接的服务 |
| **不能删除** | 日历事件在未经你确认的情况下不会被删除 |
| **不能共享** | 你的数据不会与其他用户、品牌方或第三方共享 |

## 你始终掌握主动权

- **随时断开连接** — 随时从你的控制台移除 Google、Notion 或任何连接
- **删除机器人** — 删除你的 Telegram 机器人会同时清除你的沙箱环境和所有相关数据。我们的服务器不存储你的个人敏感数据。
- **操作需确认** — 没有你的批准，任何邮件不会被发送，任何日历事件不会被创建
- **行动透明可见** — Kairos 始终告诉你它在做什么，以及为什么这么做

## 我们申请的权限

### Google

| 权限 | 用途 |
|---|---|
| 读取邮件（gmail.readonly） | 分类识别品牌合作邮件和重要邮件，并通知你 |
| 发送邮件（gmail.send） | 发送你批准的回复 |
| 日历事件（calendar.events） | 创建和管理你的日程安排 |
| 基础身份（openid, email） | 识别你的账户 |

不申请访问 Google 云端硬盘、相册、通讯录或其他 Google 服务。

### Notion

Kairos 仅申请访问你在连接过程中明确共享的页面和数据库。

### Telegram

你的机器人令牌仅用于通过你的个人机器人与你沟通。它经过加密，存储在你的隔离环境中。
