## あなたのデータは、あなたのもの

Kairos にはメール、ブランド案件、契約書、財務情報など、デリケートな情報をお任せいただいていることを私たちは深く認識しています。その責任を真剣に受け止めています。

## データの保護方法

### 完全に分離された環境

すべてのユーザーに完全にプライベートなサンドボックス環境が用意されています。あなたのデータ、認証情報、AIマネージャーの状態は、他のすべてのユーザーから完全に分離されています。ストレージの共有やクロスユーザーアクセスはありません。

### 暗号化

すべての認証情報（Gmail トークン、Notion トークン、ボットトークン）は **AES-256-GCM** で暗号化されています — これは銀行や金融機関が使用するのと同じ暗号化規格です。認証情報は保存時に暗号化され、承認された操作のために必要なときだけ復号されます。

### メールのプライバシー

新しいメールが届くと、Gmail は Kairos にメールの内容ではなく、IDナンバーだけを含む通知を送ります。Kairos はこの ID をあなたのプライベート環境に転送し、AIマネージャーが Gmail から直接メールを読み取ります。私たちのサーバーは二重処理を防ぐために各通知の短い ID のみを保存します。メールの件名、本文、添付ファイル、送信者情報が私たちのデータベースに触れることはありません。

### Telegram のセキュリティ

あなたの Telegram ボットは BotFather を通じてあなた自身が作成・所有します。Kairos はすべての操作でボットのオーナー ID を通じてあなたの身元を確認します — グループチャットでも、他の誰もあなたのAIマネージャーにアクセスすることはできません。

## Kairos ができることとできないこと

| | 詳細 |
|---|---|
| **読み取り可能** | メール（Gmail）、カレンダーイベント、アクセスを許可した Notion ページ |
| **作成可能** | カレンダーイベント、Notion ページ、メールの下書き |
| **送信可能** | メール — あなたの明示的な確認後のみ |
| **アクセス不可** | Google Drive、Google フォト、または連携していないサービス |
| **削除不可** | カレンダーイベントはあなたの確認なしに削除されません |
| **共有不可** | あなたのデータは他のユーザー、ブランド、第三者と共有されません |

## あなたが常にコントロール

- **いつでも連携解除** — Google、Notion、その他の連携はいつでもダッシュボードから削除できます
- **ボットの削除** — Telegram ボットを削除すると、サンドボックスと関連するすべてのデータが削除されます。私たちのサーバーにあなたの個人的なデリケートなデータは保存されません。
- **確認が必要** — あなたの承認なしにメールは送信されず、イベントも作成されません
- **透明なアクション** — Kairos は何をしているのか、なぜそうするのかを常に伝えます

## リクエストする権限

### Google

| 権限 | 理由 |
|---|---|
| メールの読み取り (gmail.readonly) | ブランド案件と重要なメールを分類・通知するため |
| メールの送信 (gmail.send) | あなたが承認した返信を送信するため |
| カレンダーイベント (calendar.events) | スケジュールの作成と管理のため |
| 基本的な識別情報 (openid, email) | アカウントを識別するため |

Drive、フォト、連絡先、その他の Google サービスへのアクセスは一切ありません。

### Notion

Kairos は連携プロセス中にあなたが明示的に共有したページとデータベースへのアクセスのみをリクエストします。

### Telegram

あなたのボットトークンは、あなたの個人ボットを通じたコミュニケーションのみに使用されます。暗号化され、あなたの隔離された環境に保存されます。
