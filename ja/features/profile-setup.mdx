## AI秘書にあなたのことを知ってもらう

クリエイタープロフィールは、Kairos がブランドとの相性を評価し、料金を提案し、あなたの状況に合わせたアドバイスをするために使うものです。コンテンツとオーディエンスについてより多くを知るほど、精度の高いレコメンデーションが得られます。

## プロフィールの設定方法

Kairos がステップバイステップでプロセスをガイドします。フォームへの入力は不要 — ただ会話するだけです。

### SNS を共有する

プロフィール情報を提供する方法はいくつかあります：

- **URL を共有する** — YouTube チャンネル、X プロフィールなどのリンクを送る
- **OAuth で直接連携** — TikTok はダッシュボードからワンクリックでアカウントを連携し、データを自動同期できます
- **スクリーンショットを送る** — プロフィールページのスクリーンショットを撮ってボットに送る
- **説明する** — コンテンツ、オーディエンス、使っているプラットフォームを Kairos に話す

これらの組み合わせも使えます。一番簡単な方法を使ってください。

### 対応プラットフォーム

Kairos は YouTube、X/Twitter、TikTok、Instagram などのプロフィールを分析できます。TikTok はダッシュボードからOAuth連携でき、最も正確なデータが取得できます。その他のプラットフォームはURLやスクリーンショットで分析可能です。

## Kairos が分析する内容

プロフィールを共有すると、Kairos が多角的な全体像を構築します：

| 項目 | 分析内容 |
|---|---|
| **ニッチ** | コンテンツのカテゴリーとサブトピック |
| **エンゲージメント** | 直近のエンゲージメント率 |
| **コンテンツスタイル** | フォーマットの好み、投稿パターン、制作クオリティ |
| **トーン** | フォーマル度、ユーモア、教育的深さ、パーソナリティ |
| **オーディエンス** | フォロワー数、成長トレンド、オーディエンス属性（利用可能な場合） |

## プロフィールが必要なタイミング

案件評価を開始するたびにプロフィールが確認されます。まだ設定していない場合、Kairos は先に進む前にセットアップを促し、プロセスをガイドします。

<Tip>
  事前にプロフィールを設定しておく必要はありません。必要になったときに Kairos がプロンプトを出し、プロセスをガイドします。
</Tip>

## マルチプラットフォーム対応

複数のプラットフォームで活動している場合、Kairos はすべてのデータを統合して1つのクリエイタープロフィールを作成します。これにより、ブランド評価においてあなたのリーチとコンテンツスタイルの完全な全体像が得られます。

## プライバシー

プロフィールデータはあなたのサンドボックスにプライベートに保存されます。案件評価のためだけに使用され、ブランドや他のユーザーと共有されることはありません。
