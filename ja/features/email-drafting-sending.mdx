## メール作成を、もっとスムーズに

ブランドへの返信、支払いのフォローアップ、コールドピッチの送付など — Kairos はあなたのトーンとスタイルに合ったメールを下書きします。あなたの許可なしに送信されることは一切ありません。

## 仕組み

### 下書きを依頼する

Kairos にメールの下書きを依頼する方法はいくつかあります：

- メール通知の **直接返信** をタップ
- _「Nike にキャンペーンのタイムラインについてフォローアップメールを書いて」_ と依頼
- 支払い通知の **リマインダーを送る** をタップ
- _「あのスキンケアブランドへのお断りメールを書いて」_ と伝える

### 送信前にレビュー

すべてのメール下書きは LINE または Telegram に3つの選択肢とともに表示されます：

- **送信** — 表示されたままメールを送信
- **編集** — 変更したい点を伝えると、Kairos が修正します
- **キャンセル** — 下書きを完全に破棄

<Tip>
  必要なだけ何度でも編集できます。変更したいことを伝えるだけです：_「もっと短くして」_、_「もっと熱意を込めて」_、_「来週空いていることを追加して」_。
</Tip>

## 敬語メール

日本のビジネスシーンで欠かせない敬語。Kairosは、相手との関係性や文脈に応じて適切な敬語レベルを自動で選択し、ビジネスマナーに沿ったメールを作成します。

- **クライアントへの初回連絡**: 最も丁寧な敬語（謙譲語 + 尊敬語）を使用
- **既存の取引先へのフォローアップ**: 適度にフォーマルな丁寧語を使用
- **社内や親しい関係**: カジュアルすぎない程度の敬語に調整

季節の挨拶（時候の挨拶）も文脈に応じて自動で追加されます。もちろん、トーンの調整は「もう少しカジュアルに」「もっと丁寧に」と伝えるだけで対応します。

<Tip>
  敬語に自信がなくても大丈夫。Kairosが適切な表現を選んでくれるので、あとは内容を確認するだけです。
</Tip>

## スマートなコンテキスト

Kairos の下書きはテンプレートとは違います。会話の履歴、案件の文脈、あなたのコミュニケーションスタイルを把握した上で作成されるため、毎回の下書きがあなた自身が書いたかのような内容になります。

## 下書きできるメールの種類

| ユースケース | 例 |
|---|---|
| **ブランドへの返信** | コラボ招待への返答（料金と空き状況を含む） |
| **交渉** | 料金、成果物、スケジュールのカウンターオファー |
| **フォローアップ** | 音沙汰のない案件への確認 |
| **支払い催促** | 期限超過の請求書へのプロフェッショナルなフォローアップ |
| **納品メール** | 成果物のリンクを含むコンテンツ提出 |
| **お断り** | 合わない案件へのていねいな辞退 |
| **敬語メール** | クライアントへのフォーマルな問い合わせ・お礼・依頼 |
| **時候の挨拶付き** | 季節に応じた挨拶を含むビジネスメール |

## スレッド管理

Kairos が検出したメールへの返信は、Gmail で正しくスレッド化されます — 新しいメールではなく、同じ会話内に表示されます。これにより受信トレイが整理され、ブランドも完全なやり取りの文脈を確認できます。

## プライバシーとコントロール

- **明示的な確認なしにメールが送信されることはありません**
- 下書きはその場で生成されます — 保存もログも残りません
- いつでも自分で返信を入力することができます
- Kairos がブランドに自発的に連絡することはありません
