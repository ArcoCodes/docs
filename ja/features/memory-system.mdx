## 記憶するAI秘書

<Tip>
  **Kairos は使えば使うほど、あなたの仕事の仕方をより深く理解します。** あなたの行動から好みを学び、自動的に反映するので、同じことを繰り返す時間が減り、制作に集中できる時間が増えます。
</Tip>

## Kairos が学習すること

Kairos は以下のようなことを自然に把握します：

- ブランド案件に対する好みと料金のアプローチ
- コミュニケーションのスタイルとトーン
- どの通知を重要視していて、どれを必要としないか
- スケジュールの管理の好み

## 透明性

Kairos は黙ってあなたの判断を上書きすることはありません。記憶された好みが現在の状況と矛盾する場合、その矛盾を表示してあなたに判断を委ねます：

_「通常はサプリメントブランドの案件を断っていますが、このブランドは適合スコアが高いです。それでも評価しますか？」_

## 指示を設定する

いつでも好みを明示的に設定できます：

- _「5万円を超える案件は常にディープリサーチを使って」_
- _「簡潔で直接的なメールが好き — 余計な言葉はいらない」_
- _「午前9時前に通知で起こさないで」_
- _「noreply アドレスからのメールはスキップして」_

これらは最高優先度が与えられ、学習されたパターンより優先されます。

<Tip>
  好みはいつでも更新・削除できます。Kairos に伝えるだけです：_「ニュースレターメールのフィルタリングをやめて」_ や _「やっぱりスタンダードリサーチをデフォルトに戻して」_。
</Tip>
